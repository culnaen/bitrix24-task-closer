# Bitrix24 Task Closer

Userscript для Bitrix24: добавляет к карточкам задач действие, которое записывает
затраченное время, добавляет результат и завершает задачу через REST API.

## Установка

1. Установьте расширение [Tampermonkey](https://www.tampermonkey.net/).
2. Создайте в нём новый userscript и замените его содержимое файлом
   `bitrix24-close-task.user.js` из этого репозитория.
3. В начале файла заполните объект `CONFIG`.
4. Откройте задачи на своём портале по адресу
   `https://<ваш-портал>/company/personal/user/<ID>/tasks/` и обновите страницу.

## Настройка `CONFIG`

```js
const CONFIG = Object.freeze({
    WEBHOOK_USER_ID: 26,
    WEBHOOK_CODE: 'PASTE_INCOMING_WEBHOOK_CODE_HERE',
    ELAPSED_SECONDS: 60,
    COMPLETION_COMMENT: 'Работа по задаче выполнена.',
    COMMENT_MODE: 'auto',
});
```

- `WEBHOOK_USER_ID` — ID пользователя, от имени которого создан входящий вебхук.
- `WEBHOOK_CODE` — код входящего вебхука Bitrix24. Это секрет: не публикуйте его
  и не коммитьте файл с реальным значением.
- `ELAPSED_SECONDS` — время, добавляемое к задаче, в секундах.
- `COMPLETION_COMMENT` — текст результата задачи.
- `COMMENT_MODE` — `auto` для автоматического выбора API; при необходимости можно
  указать `chat` или `legacy`.

Создать вебхук можно в Bitrix24: **Разработчикам → Другое → Входящий вебхук**.
Для работы скрипта у вебхука должны быть права на задачи и чат. Метаданные
userscript уже разрешают запуск на HTTPS-порталах Bitrix24; если у вас другой путь
к списку задач, скорректируйте строку `@match`.

## Публикация

Перед отправкой на GitHub убедитесь, что в `WEBHOOK_CODE` остаётся значение-заглушка.
Локальные варианты userscript с секретом можно хранить под именем
`*.local.user.js`: они исключены из Git.
